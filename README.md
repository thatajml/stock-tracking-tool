# QuantSense: Professional Probabilistic Stock Dashboard

QuantSense is a stock tracking tool designed to bridge the gap between retail trading and institutional-grade quantitative analysis. It transforms raw market data into actionable high-probability trading signals using a multi-factor weighted scoring engine.

## Features
- **Real-Time Data & Search**: NSE market connectivity and historical context.
- **Multi-Panel Synchronized Dashboard**: Price Action Panel, Volume Profile Panel, and Momentum & Timing Panel with various technical indicators.
- **Probabilistic Scoring Engine**: Calculates a Net Confidence Score based on Trend, Momentum, Timing, and Confirmation.
- **Algorithmic Transparency**: Provides a logic breakdown for signals.

## Technical Stack
- **Frontend**: React (Vite), Material UI (MUI), TradingView Lightweight Charts
- **Backend**: Flask (Python), yfinance, ta (Technical Analysis Library)
